# Keras Labs - Coursera & IBM

Este repositório reúne diferentes experimentos e implementações utilizando **Keras**, desenvolvidos a partir dos laboratórios dos cursos de Inteligência Artificial da **Coursera** em parceria com a **IBM**.

## Objetivo

O principal objetivo deste projeto é **aprendizado e prática**, explorando conceitos como:
- Redes neurais artificiais
- Deep Learning
- Modelagem e treinamento com Keras
- Avaliação de modelos

## Observação

Este repositório não tem fins comerciais — foi criado exclusivamente para **estudo, experimentação e evolução técnica**.
